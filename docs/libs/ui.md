# User Interface Documentation

La partie visuelle de GarantFacile repose sur une stack moderne axée sur la performance et l'esthétique "Premium".

## Technologies Clés

### 1. HeroUI (NextUI v3)

Bibliothèque de composants UI basée sur Tailwind CSS et Framer Motion.

- **Tokens de Thème** : Les couleurs et le layout sont centralisés dans `tailwind.config.js`.
- **Bordures** : Un radius standard de 14px est appliqué sur les boutons et cartes pour un aspect moderne.
- **Responsivité** : Utilisation des grilles et Flexbox natives de HeroUI.

### 2. Tailwind CSS

Framework utilitaire pour le style.

- **Customization** : Extension des couleurs `primary`, `secondary`, `success`, etc., pour correspondre à l'identité visuelle de GarantFacile.
- **Maintenance** : Suppression des styles inline au profit de classes utilitaires réutilisables.

### 3. Material Symbols

Système d'iconographie standardisé remplaçant les emojis.

- **Source** : Google Fonts (Material Symbols Outlined).
- **Usage** : Cohérence visuelle cross-plateforme et accessibilité améliorée.

## Identité Visuelle

- **Primaire** : Bleu profond (Brand).
- **Secondaire** : Nuances de bleu et d'indigo.
- **Radius** : 14px (Standard).
- **Typographie** : Sans-serif moderne via Google Fonts (Inter/Roboto).
